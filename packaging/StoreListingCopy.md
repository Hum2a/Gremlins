## Full description

**(Marketing / Store detail page — paragraphs)**

Gremlins runs from the system tray and adds light, deniable chaos while you work—never destructive, always easy to blame on “the universe.”

**What you get**

- **Tray-first dashboard** — Open the app when you want it; otherwise it stays out of the way.
- **A roster of gremlins** — Each personality does something different: tiny cursor drift, occasional character swaps, clipboard surprises, gentle audio reactions to distracting titles, fake input lag bursts, slow window nudges, and more.
- **Severity that scales** — From slower and subtler to faster and wilder, tuned per gremlin.
- **Rules & timing** — Quiet hours, optional daily windows, focus-app pauses, idle boost, soft start, and pauses during presentations or fullscreen.
- **Panic & resume** — Silence everything instantly when you need peace; bring tricks back without reinstalling.
- **Themes & polish** — Dark-first UI that fits Windows 11; optional UI sounds.

Everything runs **on your PC**. Hooks are used **only** for tricks you enable—nothing is uploaded to Gremlins servers.

Use responsibly on machines you own or where everyone consents to the joke.

## Notes for certification

*(Private — helps reviewers; not shown on the Store listing.)*

Gremlins is a desktop tray application for Windows. On launch it shows or skips onboarding, then runs from the notification area. Users enable discrete “gremlin” features that may use low-level input hooks **only when those features are toggled on**. Users can **Panic** to silence all tricks without uninstalling, **Resume** to restore behavior, or quit via **Exorcise** from the tray menu.

No cloud account is required for core features. Optional GitHub update checks may contact GitHub if the user configures a repo URL in settings—otherwise no network dependency for core tricks.

Please exercise hooks/tricks only with reviewer-safe timings or after disabling tricks in the dashboard if automation interferes with UI automation tests.
